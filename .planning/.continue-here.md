# Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| `filepath.ToSlash` host-OS test trap | A test asserted a backslash Windows path (`C:\…\.windsurf\…`) through `hostFromExtRoot` (which uses `filepath.ToSlash`). ToSlash only rewrites the HOST OS separator, so it passed on local Windows `go test` but FAILED `ci.yml` Test on ubuntu/macos (backslashes stay literal there). Turned `main` CI red. | advisory | Never assert backslash paths through ToSlash-based code in cross-platform tests. Local Windows `go test` will not catch it; only the Linux/macOS CI legs do. |
| LlamaFirewall sidecar is CI/Linux-only | `codeshield`→`semgrep` has no native Windows build (semgrep#1330); `beekeeper llamafirewall install` aborts on native Windows. | advisory | Don't try to run the full LLMF install/e2e on the Windows dev box; the gated-model DOWNLOAD is the only Windows-verifiable part. |

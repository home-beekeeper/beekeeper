| three catalog sources → aggregator | Each source is independently untrusted; corroboration is the trust mechanism |
| hook handler → external registries (OSV/Socket) | Hot-path HTTP must be deadline-bounded and degrade safely |
| audit record → operator | The audit record is the forensic trail; provenance must be complete and tamper-evident in content |

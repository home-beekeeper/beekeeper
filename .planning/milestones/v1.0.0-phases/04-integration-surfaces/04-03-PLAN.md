## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| MCP client → gateway HTTP | Every byte from client is untrusted; token required before any processing |
| gateway → upstream MCP server | After policy allow; gateway trusts the upstream response |
| upstream response → MCP client | Warn path injects _beekeeper_warning; all other responses forwarded verbatim |
| state.json → gateway token read | Other local processes must not read the token |

| T-04-03-02 | Elevation of Privilege | Timing attack on token comparison | mitigate | crypto/subtle.ConstantTimeCompare in verifyToken; grep gate in acceptance criteria |

SO_PEERCRED pattern (from RESEARCH.md §4.2 — VERIFIED):
```go
func verifyPeerUID(conn net.Conn, expectedUID uint32) error {
    unixConn, ok := conn.(*net.UnixConn)
    if !ok { return fmt.Errorf("not a Unix connection") }
    rawConn, err := unixConn.SyscallConn()
    if err != nil { return fmt.Errorf("getting raw conn: %w", err) }
    var ucred *unix.Ucred
    var innerErr error
    ctrlErr := rawConn.Control(func(fd uintptr) {
        ucred, innerErr = unix.GetsockoptUcred(int(fd), unix.SOL_SOCKET, unix.SO_PEERCRED)
    })
    if ctrlErr != nil { return fmt.Errorf("control: %w", ctrlErr) }
    if innerErr != nil { return fmt.Errorf("getsockopt SO_PEERCRED: %w", innerErr) }
    if ucred.Uid != expectedUID { return fmt.Errorf("peer UID %d does not match expected %d", ucred.Uid, expectedUID) }
    return nil
}
```

    Create internal/ipc/server.go with build tag `//go:build linux || darwin`:

    ```go
    type Server struct {
        listener  net.Listener
        sockPath  string
        ownerUID  uint32
    }

    // Handler is called with an authenticated connection. It is responsible for
    // reading one IPCCommand and writing one IPCResponse, then returning.
    type Handler func(conn net.Conn)

    func NewServer(sockPath string, ownerUID uint32) (*Server, error)
    func (s *Server) Serve(ctx context.Context, handler Handler) error
    func (s *Server) Close() error
    ```

    NewServer implementation:
    - Remove any stale socket file at sockPath (os.Remove, ignore ErrNotExist)
    - net.Listen("unix", sockPath)
    - os.Chmod(sockPath, 0600)
    - os.Lchown(sockPath, int(ownerUID), -1) — pass -1 for gid to keep existing gid
    - Return &Server{listener, sockPath, ownerUID}

    Serve implementation:
    - In a loop: listener.Accept() → goroutine → verifyPeerUID(conn, s.ownerUID); if fail: conn.Close(); log; continue; else call handler(conn)
    - Return when ctx.Done() fires: close listener, drain goroutines
    - Use `verifyPeerUID` as a package-private function following the exact RESEARCH §4.2 pattern with rawConn.Control + unix.GetsockoptUcred

    Create internal/ipc/client.go with build tag `//go:build linux || darwin`:

    ```go
    func Connect(sockPath string, timeout time.Duration) (net.Conn, error)
    func SendCommand(conn net.Conn, cmd IPCCommand, timeout time.Duration) error
    func ReadResponse(conn net.Conn, timeout time.Duration) (IPCResponse, error)
    ```

    Connect: `net.DialTimeout("unix", sockPath, timeout)` — returns (conn, err) directly.

    SendCommand: set write deadline via conn.SetWriteDeadline(time.Now().Add(timeout)), then call Encode(conn, cmd).

    ReadResponse: set read deadline via conn.SetReadDeadline(time.Now().Add(timeout)), then Decode(conn, &resp), return resp.

    Create internal/ipc/proto_fuzz_test.go with build tag `//go:build linux`:

    - Fuzz target: call `Decode(bytes.NewReader(data), &IPCCommand{})` — must not panic regardless of input; ErrMessageTooLarge and io errors are acceptable returns
    - Also fuzz Encode side: f.Add with a random string for RuleID field — call Encode on the constructed command — must not panic

    Note: the fuzz test file has `//go:build linux` so it runs in CI on Linux; the types it references (Decode, IPCCommand) are in proto.go which has no build tag and compiles everywhere.

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CLI process → Unix socket | Untrusted peer; must verify UID before processing any command |
| Network byte stream → IPCCommand | Malformed length prefixes, oversized payloads, invalid JSON must not panic or allocate unbounded memory |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01-01 | Elevation of Privilege | IPC server accepting commands from non-owner UID | mitigate | SO_PEERCRED UID check on every accepted connection via GetsockoptUcred; connection closed immediately on mismatch before any bytes are read from the client |
| T-05-01-02 | Denial of Service | Oversized IPC message exhausting daemon heap | mitigate | 64KB hard cap in Decode; ErrMessageTooLarge returned before allocation; fuzz test covers length prefix overflow |
| T-05-01-03 | Tampering | Malformed length prefix causing io.ReadFull to read wrong slice bounds | mitigate | io.ReadFull used for both length read (4 bytes) and payload read (length bytes); errors propagated, no unsafe indexing |
| T-05-01-04 | Information Disclosure | Socket file world-readable allowing any local process to query daemon state | mitigate | chmod 0600 + lchown to installingUID in NewServer; only the installing user's processes can connect |
| T-05-01-05 | Tampering | Stale socket file from crashed daemon blocking new server bind | mitigate | os.Remove(sockPath) before net.Listen in NewServer; ErrNotExist is silently ignored |
</threat_model>

## Milestone Summary

**Key Decisions:**
- Internal milestone number **v1.5.0** decoupled from the public release **v1.1.0** to avoid colliding with the parked v1.1.0 "Pollen" GSD milestone.
- Default posture = **warn + fail-soft** at the hook (ratified at Gate 1). The unknown path (missing timestamp / registry error / timeout) stays warn even under block mode.
- **IPOVR-03** (opt a rule up to block) pulled into v1.0 by maintainer decision at Gate 1; the finer per-ecosystem/per-project matrix stays roadmap.
- **Shim kept + made real + documented experimental** (not deleted) ‚Äî repointed onto `beekeeper check`; bypassable by absolute path, requires PATH prepend.
- **Schema-freeze:** KEEP the typed `posture_override_*` audit fields (audit-only, never routed to the corpus StoreSink; the frozen CorpusSchemaVersion 1.0 moat schema is intact). Re-encoding into the corpus is a deliberate future-milestone step.
- Posture is a warn-only, most-restrictive-merged, structurally-isolated layer that can NEVER lift a catalog/SPATH/self-protection block (T-09-31) ‚Äî verified by the release-gate security review.

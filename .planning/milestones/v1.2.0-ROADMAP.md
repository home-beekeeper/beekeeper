## Overview

Close the three runtime-enforcement gaps that live `beekeeper check` validation exposed ‚Äî with the agent itself as the test subject ‚Äî each locked in by a behavioral test suite proving the wiring is live:

- **F1** ‚Äî a *critical*-severity malware package (`ai-figure` / Shai-Hulud, OSV `MAL-2026-4126`) that **warned instead of blocking**.
- **F2** ‚Äî credential files (`~/.aws/credentials`, `~/.ssh/id_rsa`, `.env`, MCP configs) that returned **ALLOW** to agent reads.
- **F3** ‚Äî `pnpm`/`bun` installs that **bypassed catalog matching** entirely (unparsed).

`internal/policy` (and the new `internal/nudge` / `internal/pkgparse`) stayed pure function libraries throughout; all I/O lives in adapters. Self-defense for this milestone is the behavioral test suite itself ‚Äî every fix ships a regression test that fails on the pre-fix code, gated by `-tags fuzz` + `-tags e2e` release batteries.
